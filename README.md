Solo Traveller Recommendation System
AI-Powered Travel Companion for Personalized, Safe & Smart Solo Trips

🚀 Overview

Solo Traveller is a full-stack travel assistant designed specifically for solo travelers.
It helps users discover new destinations, connect with other solo travelers, find safe routes, explore events, and get personalized recommendations — all powered by location data, user interests, and smart backend logic.

This project includes a Flask Python backend + MySQL database + front-end website + real-time chat support.
